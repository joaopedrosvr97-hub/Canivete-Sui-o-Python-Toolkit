# Política de Segurança do Canivete Suíço Toolkit

Este projeto lida com comandos de administração de sistema, tornando a segurança uma prioridade. Estamos comprometidos em corrigir vulnerabilidades prontamente.

## Relatando uma Vulnerabilidade

Se você descobrir uma falha de segurança no código, pedimos que **NÃO** abra uma Issue pública.

Em vez disso, por favor, envie os detalhes da vulnerabilidade diretamente para o mantenedor principal do projeto (João Pedro) através do e-mail:

**[joaopedrosvr97@gmail.com]**

Faremos o possível para reconhecer, investigar e corrigir o problema o mais rápido possível e informaremos os usuários sobre qualquer correção crítica.
