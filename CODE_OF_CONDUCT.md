# Código de Conduta

Nós adotamos o Contributor Covenant, versão 2.1, como nosso Código de Conduta, e esperamos que todos os participantes sigam estas regras.

## Nossos Padrões

Exemplos de comportamento que contribuem para criar um ambiente positivo incluem:

* Usar linguagem inclusiva.
* Respeitar diferentes pontos de vista e experiências.
* Aceitar críticas construtivas.
* Focar no que é melhor para a comunidade.

## Comportamento Inaceitável

Exemplos de comportamento inaceitável incluem:

* Uso de linguagem ou imagens sexualizadas, e atenção ou avanços sexuais de qualquer tipo.
* Comentários depreciativos, insultos pessoais ou ataques *ad hominem*.
* Assédio ou bullying.

## Aplicação

Violações do Código de Conduta podem ser reportadas entrando em contato com os mantenedores do projeto no e-mail listado no `SECURITY.md`. Os mantenedores se comprometem a investigar e responder.

O não cumprimento do Código de Conduta pode levar à remoção temporária ou permanente do projeto.
